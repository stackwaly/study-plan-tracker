# 📚 Rastreador de Plano de Estudos

Aplicação web para organizar e acompanhar o progresso dos estudos ao longo do tempo.

---

## 🚀 Sobre o projeto

O **Rastreador de Plano de Estudos** foi desenvolvido para ajudar estudantes a manterem consistência e organização na jornada de aprendizado.

Com uma interface simples e intuitiva, é possível acompanhar o progresso semanal e mensal de forma visual.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (Flexbox, Variáveis CSS)
* JavaScript (DOM e LocalStorage)

---

## 📱 Funcionalidades

* ✅ Marcar semanas como concluídas
* 📊 Barra de progresso automática
* 💾 Salvamento de dados com localStorage
* 🌙 Alternância entre tema claro/escuro
* 🔍 Busca por conteúdo
* 📅 Organização por meses
